if, else if, else
_________________________________________


연산 부분에서 if, else if, else 를 사용하는 방법을 알았습니다

이번에는 if, else if, else 가 어떤관계인지 알아보도록 하겠습니다

조건 이시작 할때 (if) 를 사용 합니다 그 밑에는 (else) 가기전의 단계 else if 가있고 else 가 있습니다
그 사이에는 절대 코드같은게 와서는 안됩니다 무조건 연결이 되어야 합니다

if(조건)
{


}
else if(조건)
{


}
int value = 10; //사이에 끼면 안됩니다 무조건 조건이 끝난 시점에 코드를 작성해야 합니다
else
{


}

사이에 껴도 되는 상황은 밑에 else 가 없거나 혹은 else if 가 없을때 입니다

if(조건)
{


}
int value = 10; //사이에 껴도 괜찮습니다 왜냐하면 밑에 else 도 없고 else if 도 없기 때문입니다
if(조건)
{


}
int value = 10; //사이에 끼면 안됩니다 왜냐하면 밑에 else if 가 있기 때문에 조건이 아직 끝나지 않았습니다
else if(조건)
{


}


요약



재가 살아가면서 한 실수:

if(조건1)
if(조건2)
else if(조건2)
else if(조건2)

원래는 

if(조건1)
else if(조건1)
if(조건2)
else if(조건2)

이렇게 되어야 하는데..


정리하자면 if 와 else if, else 모두 다 친척이다, 다 붙어있어야 하고 사이에 코드 같은게 끼면 안된다

switch
_________________________________________


switch 는 한값에 대한 연산 입니다 그래서 if 문 보다 빠릅니다

switch (값)
{

case 값:
{

}

case 값:
{

}

case 값:
{

}

case 값:
{

}

.
.
.

default:
{


}


}

예제:

switch(40)
{

case 30:  //switch 안에 있는 값이 40 이고 30 과 같지 않으니 다른 케이스로 넘어갑니다
{


}

case 40:  //switch 안에 있는 값이 40 이고 40 과 같으니 case 40: 블락 안에 있는 코드를 실행 시킵니다
{

}

default:  //만약 모든 case를 걸쳐도 같지 않으면 default: 블락 안에 있는 코드를 실행 실행 시킵니다
{

}


}
